Grundlæggende Web

Teknologier og Værktøjer

  • Gestaltprinciper
  • CSS Grid
  • Responsivt design

Gestaltprincipper: Vi arbejdede med designprincipper som Nærhed, Lighed, Fælles skæbne og Lukkethed til at skabe orden og visuel balance.

Det har jeg brugt som eksempel i navigation bar hvor der er lige afsand mellem elementerne så loven kontinuitet er holdt her. Vi har også lovene om nærhed og lighed her. Det skaber visuel klarhed.

CSS Grid: Gridsystem: Et grid opdeler layoutet i rækker og kolonner og skaber struktur. Vi brugte CSS Grid til at placere indhold i et ensartet og fleksibelt system.

Ved brug at grid system fik jeg billederne til at stå ifølge wireframen. Man bryger det primært til desktop layout. De skaber struktur og orden i layoutet.
